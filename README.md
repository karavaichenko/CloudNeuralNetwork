# Проект Cloud Neural Network
Это репозиторий backend-составляющей проекта Cloud Neural Network - платформы для обучения нейронным сетям.
Ссылка на репозиторий frontend-составляющей: https://github.com/prohanter34/CloudNeuralNetworkFrontend

## Как запустить?
1. Склонируйте репозиторий
2. Установите зависимости проекта из файла requirements.txt
3. Запустите файл server/main.py
